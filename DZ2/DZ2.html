Задача №1

Дан код:

var a = 1, b = 1, c, d;

c = ++a; alert(c); // 2
Ответ: Эта операция выполняет инкремент, прибавляет 1 операнда и возвращает его значение. Если используется префикс, возвращается значение, полученное после выполнения инкремента. Вывод "c" в модальном окне.

d = b++; alert(d); // 1
Ответ: Инкремент постфикс, возвращается значение, которое было до выполнения инкремента. Вывод "d" в модальном окне.

c = (2+ ++a); alert(c); // 5
Ответ: Сначала выполняется инкремент префикс, увеличение a=a+1, а потом уже с новым значением "a" другие операции. Т.е. 2 + ((a=2)+1) присвоить c. Вывод "c" в модальном окне.

d = (2+ b++); alert(d); // 4
Ответ: Сначала выполняется операция инкремент постфикс b=b+1, но сложение происходит с предыдущим значением "b", т.е. 2 + 2 = 4 присвоить d. Вывод "d" в модальном окне.

alert(a); // 3
Ответ: Значение "a" увеличивалось в предыдущих операциях инкремента 2 раза на 1, т.е. a=(a=1)+1+1

alert(b); // 3
Ответ: Значение "b" увеличивалось в предыдущих операциях инкремента 2 раза на 1, т.е. b=(b=1)+1+1

Задача №2

Чему будет равен x в примере ниже?
var a = 2;
var x = 1 + (a *= 2);
Ответ: x будет =
В выражении var x = 1 + (a *= 2) применена краткая форма записи арифметического оператора "*=", которая соответствует полной форме выражения: var x = 1 + (а = а * 2); - значение переменной "x", согласно арифметическим действиям, будет равно "5".
